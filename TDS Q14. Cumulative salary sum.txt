14. Cumulative salary sum
The following table holds monthly salary information for several employees. Write a query
 to get, for each month, the cumulative sum of an employee’s salary over a period of 3 months, 
excluding the most recent month. The result should be ordered by ascending employee ID and month.

#### CLEVER SOLUTION WHERE THEY USED DESC IN RANK

select id, pay_month , salary , sum(salary) OVER( partition by id order by month_rank DESC) from

(select *, rank() OVER(partition by id order by pay_month DESC) as month_rank from employee)T1
where month_rank != 1
and month_rank <=4
order by id, pay_month

Another way:

Select id, pay_month , salary , sum(salary)OVER (partition by id order by pay_month ROWS BETWEEN 2 PRECEEDING AND CURRENT ROW ) as cumsalary
FROM
(Select id, pay_month , salary, dense_rank() OVER( partition by id order by pay_month DESC) as rnk
From employee) t1

Where rnk != 1
Order by id, pay_month

Python

df[‘rank’]=sort_values(by=[‘id’,’pay_month’], ascending=[True, False]).groupby([‘id’]).cumcount()+1
df[‘cumsum_salary’]=df.l‘salary’].cumsum()

df_final=df.loc[df[‘rank’]!=1 & df[‘rank’]<=4, [‘id’,’pay_month’,’salary’,’cumsum_salary’]]
df_final_sorted=df_final.sort_values(by=[‘id’,’pay_month’], ascending=[True, True])
