18. Maximally-separated cities
From the same table in question 17, write a query to return the 
furthest-separated pair of cities for each state, and the corresponding 
distance (in degrees, rounded to 2 decimal places) between those two cities

Select state, city_1, city_2, distance from
(
Select state, city_1, city_2, distance , dense_rank()OVER( partition by state order by distance DESC) as rnk
From
(
Select state, c1.city as city_1, c2.city as city_2 ,  sqrt(pow((c1.lat-c2.lat),2) + pow((c1.lon-c2.lon),2) ) as distance from stations c1
Join stations c2
On c1.state=c2.state
And c1.city!=c2.city
) t1
) t2
Where rnk =1

Python

df_merged=stations.merge(stations, how=‘inner’, on=‘state’)
df_merged = df_merged.loc[df_merged[‘city_x’]!=df_merged[‘city_y’],:]
df_merged[‘distance’]= np.sqrt((df_merged[lat_x]- df_merged[lat_y])**2 + (df_merged[lon_x]- df_merged[lon_y])**2 )
df_merged[‘distance’]=df_merged[‘distance’].round(2)

df_merged[‘rank’]=df_merged.sort_values(by=[‘state’, ‘distance’], ascending=[True, False]).groupby(‘state’).cumcount()+1

df_final=df_merged.loc[df_merged[‘rank’]==1, [‘state’,’city_x’,’city_y’,’distance’]]

df_final= df_final.rename(columns={‘city_x’:’city_1’, ‘city_y’:’city_2’})
