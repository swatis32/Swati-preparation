2. Changes in net worth
From the following table of transactions between two users, write a query to return the change in net worth for each user, ordered by decreasing net change.
My solution:


Select user, net_worth from

(
Select user, transaction_date, sum(amount)over(partition by user order by transaction_date) as net_worth ,
dense_rank()over( partition by user, order by transaction_date DESC) as rnk

FROM

(Select transaction_date, sender as user, -1* amount as amount from transactions
Union all
Select transaction_date, receiver as user, amount as amount from transactions )t1
) t2 
Where rnk=1
Order by net_worth DESC
