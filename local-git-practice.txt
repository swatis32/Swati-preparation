Generate token on github.com using these steps:
Generate a Personal Access Token (PAT)Log in to your account on the GitHub Website.
Click your Profile Photo in the top right corner and select Settings.
Scroll down the left sidebar and click Developer settings.
Click Personal access tokens and select Tokens (classic).Click Generate new token \(\rightarrow \) Generate new token (classic).
Give it a descriptive Note (e.g., "Terminal access") and choose an Expiration date.
Select the repo checkbox (this allows you to push and pull).Click Generate token at the bottom of the page.Copy the token immediately. You will not be able to see it again.
